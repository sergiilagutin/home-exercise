# Home exercise

Minimum / maximum path in triangle

## Tech stack:
* Scala 2.13
* cats, cats-effect
* specs2

## Read from standard input
execute `sbt run`

## Read from file
You can specify absolute or relative path

execute `sbt "run <file_path>"`

## Run tests
execute `sbt test`

## Format code
execute `sbt scalafmt`
